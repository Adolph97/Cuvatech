## Run Locally

**Prerequisites:** Node.js

# Setup Instructions

This is the setup flow for the cuva tech portfolio repo (v1) which is hosted at [Cuva tech](https://cuvatech.com).

Before anything, clone the repo locally via `git clone github repo link` then cd into repo.

Ensure that

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

> Open a different terminal and run the backend server
> **Backend**

1.

```bash
pnpm run server
```

1. Then go to `localhost:3000/admin` to access the admin dashboard.
